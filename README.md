# SyncPro (교육청 IoT 충전함 관리 서비스)

교육청, 교육지원청, 학교 단위로 보급되는 **IoT 태블릿·휴대기기 충전함**을  
실시간으로 관리·제어할 수 있는 **관제 솔루션 & 선생님용 모바일 앱 서비스**입니다.

본 프로젝트는 공공기관 단위 디바이스 관리의 효율성과  
운영 편의성을 높이기 위해 구축되었습니다.

---

# 주요 기능 요약

## 네트워크 & 디바이스 등록 구조

충전함은 **학교 내부 Wi-Fi 환경**에 연결되어야 관제 서버와 통신이 가능합니다.  
이를 위해 설치 담당자 또는 교사가 다음 절차로 기기를 등록합니다.

1. **학교 AP 선택**  
   - 현장 설치 시, 앱에서 학교 내부 Wi-Fi 리스트를 조회하여 선택

2. **Wi-Fi 보드를 통한 연결 설정**  
   - 충전함 내부의 Wi-Fi 모듈이 AP 정보(SSID / PW)를 수신하여 연결 시도

3. **기기 등록 (Device Binding)**  
   - 앱에서 해당 충전함의 **시리얼 번호(고유 식별값)** 를 입력 또는 QR 스캔
   - 서버에 충전함 ⇔ 학교 ⇔ 교육지원청 계층 구조로 매핑됨

4. **관제 서버 연결**  
   - MQTT 및 REST API 기반으로 **상태 정보 / 제어 명령** 통신 개시

> 이 등록 구조를 통해 각 충전함은 **소속 학교를 기준으로 독립적으로 운영**되며,  
> 관리자는 상위 기관(교육청/교육지원청 단위)에서 일괄 모니터링이 가능합니다.

# 관리자 관제 시스템 (.NET MVC)
| 기능 | 설명 |
|---|---|
| 충전함 생산 및 출고 관리 | 공장 생산 → 시리얼 발급 → 배포 이력 관리 |
| 배송 및 설치 현황 관리 | 교육청 / 교육지원청 / 학교별 설치 현황 관리 |
| 기기 상태 모니터링 | 온도, 전원 상태, 충전 유닛 상태, 네트워크 상태 실시간 조회 |
| 원격 제어 기능 | 충전함 전원 제어, 윤번(순차충전) 기능 제어 |
| 고장 / AS 처리 | 장애 접수 → 처리 이력 관리 |

### 교사 / 현장 담당자 앱 (React Native)
| 기능 | 설명 |
|---|---|
| 실시간 충전 상태 조회 | 충전 포트별 상태, 연결 여부, 온도 확인 |
| 원격 전원 On/Off | 현장에서 앱으로 기기 제어 가능 |
| 알림 기능 | 충전함 과열 / 충전 이상 시 알림 제공 |

## 데이터 구조

충전함에서 발생하는 실시간 데이터는 **MQTT 기반 소켓 프로토콜**로 서버에 전달되고,  
**MongoDB**에 Raw 형태로 저장됩니다.  
관리자/선생님 화면에서는 이를 가공하여 제공합니다.
